Add a country selector to your Strong Testimonials form.

== Description ==

Add a country selector to your Strong Testimonials form. Requires Strong Testimonials version 2.28.3 or higher.

== Installation ==

1. Go to Plugins > Add New.
1. Search for "strong testimonials country selector".
1. Click "Install Now".

OR

1. Download the zip file.
1. Upload the zip file via Plugins > Add New > Upload.

Activate the plugin.

== Frequently Asked Questions ==

= How do I add the country selector to the form? =

In the fields editor, add a new field. Select "country selector" as the field type.
